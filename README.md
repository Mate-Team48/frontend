## Frontend part of this website

### Technology stack:
 - React
 - TypeScript
 - HTML
 - SCSS

### How to run this project:
1. Download Node.js 20.11.1 [here](https://nodejs.org/en/blog/release/v20.11.1)
2. `cd ./frontend`
3. `npm install`
4. `npm run dev`
