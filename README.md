## Frontend part of this website

### Technology stack:
 - React
 - TypeScript
 - HTML
 - SCSS

### How to run this project:
1. Download Node.js 18.17.1 [here](https://nodejs.org/en/blog/release/v18.17.1)
2. `npm install`
3. `npm run dev`
